### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Create all data
   ```sh
   make all
   ```
3. Select date range
   ```sh
   make jan 2021 dec 2022
   ```
   ```sh
   make jan dec 2021 2022
   ```
   ```sh
   make jan dec 2022
   ```
4. Select specific month | year
   ```sh
   make {year} or make {month} {year}
   ```
